# Background

The data analytics field continues to grow rapidly as organizations increasingly rely on data-driven decision making. This project was created to analyze trends in the data job market and identify:

- The highest paying skills for Data Analysts
- The most in-demand technical skills
- The relationship between salary and demand
- Industry trends shaping the future of analytics

The analysis uses SQL to explore datasets related to job postings, salaries, and required skills.

# Tools I used

- **SQL** – For querying and analyzing the data
- **PostgreSQL** – Database management system used for storing and querying data
- **pgAdmin** – Managing and interacting with PostgreSQL databases
- **VS Code** – Writing SQL scripts and project documentation
- **Git & GitHub** – Version control and project sharing

# The Analysis

This project focuses on several key areas of the data analytics job market:

### 1. Top Paying Skills

Analyzed the highest-paying technical skills associated with Data Analyst roles.

Some of the top-paying skills identified include:

- Golang
- Terraform
- Kafka
- PyTorch
- TensorFlow
- Hugging Face

### 2. Most In-Demand Skills

Examined the skills most frequently requested in job postings.

The most in-demand skills included:

- SQL
- Excel
- Python
- Tableau
- Power BI

### 3. Salary vs Demand

Compared highly demanded skills with high-paying skills to identify valuable career opportunities.

The analysis showed that:

- SQL remains the most essential skill
- Python continues to grow in importance
- AI and machine learning skills command significantly higher salaries
- Cloud and automation skills are becoming increasingly valuable

### 4. Emerging Industry Trends

The project also revealed that modern data analyst roles are becoming more technical and engineering-oriented.

Companies increasingly value analysts who can:

- automate workflows
- build data pipelines
- work with cloud infrastructure
- integrate AI tools into analytics processes

This project highlights the evolving nature of the data analytics field. While foundational skills like SQL and Excel remain important, higher-paying opportunities increasingly require:

- programming knowledge
- automation skills
- cloud technologies
- machine learning familiarity

The findings suggest that Data Analysts who continue building technical and engineering skills position themselves for stronger career growth and higher salaries.
